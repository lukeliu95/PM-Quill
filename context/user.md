# 目标用户画像

## 主要用户：独立创业的技术人

**名字（虚拟）：** 小王
**背景：** 前 SaaS 公司后端工程师，Agent时代决定自己做产品
**技能：** 会写代码，会用 Cursor/Claude Code，但从没做过产品经理
**痛点：**
- 有很多想法，但不知道哪个值得做
- 直接开始写代码，做到一半发现方向不对
- 不会写 PRD / Spec，每次都是"想到哪写到哪"
- 一个人没有产品经理帮忙把关

**典型场景：**
1. "我想做一个 AI 驱动的 XX"→ 不知道从哪开始
2. 看到一个有趣的开源项目 → 想基于它做产品 → 但不确定市场
3. MVP 做完了 → 不知道下一步优先做什么

## 次要用户：Vibe Coder

**背景：** 非技术背景，靠 AI 编码工具（Cursor、v0）做产品
**特点：** 完全不会写代码，但有产品感觉
**痛点：** AI 能帮我写代码，但没人帮我想清楚要写什么
